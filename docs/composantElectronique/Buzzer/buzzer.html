Ceci est un buzzer

définition :

Un buzzer est un composant électronique qui produit un son lorsqu'il est alimanté électriquement. On l'utilise souvent comme signal sonore dans des circuits électroniques pour indiquer un état ou une alarme.

★

Caractéristiques :

  • tension : Souvent 3V,5V ou 12V
  • courant consommé : Généralement faible
  • niveau sonore : Souvent entre 70 et 100 dB
★

Application :

  • Alarme et avertisseurs
  • Indicateurs sonores dans les appareils électroménagers
  • Projet arduino / Raspberry Pi
  • Jouets électroniques
  • Dispositifs médicaux pour alerter
